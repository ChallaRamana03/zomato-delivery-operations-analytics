# Zomato Delivery Operations Analytics

## Business Problem

Food delivery companies need to monitor delivery performance, operational efficiency, and customer service quality to ensure timely deliveries and improve customer satisfaction. This project analyzes delivery operations data to identify factors affecting delivery performance and generate actionable business insights.

## Project Objective

* Evaluate delivery performance across cities and vehicle types.
* Monitor operational KPIs and delivery efficiency.
* Analyze the impact of traffic, weather, and festivals on delivery time.
* Build a reporting dashboard to support data-driven decision-making.

## Dataset Information

* Dataset: Zomato Delivery Operations Dataset
* Records: 45,584
* Tool Used: Microsoft Excel

## Data Cleaning

* Removed duplicate records
* Standardized data formats
* Created analysis-ready dataset
* Applied data validation and transformation techniques

## KPIs Tracked

* Total Orders
* Average Delivery Time
* Average Delivery Partner Rating
* Delayed Orders
* Traffic Impact Analysis
* Weather Impact Analysis
* Festival Impact Analysis

## Key Insights

* Metropolitan cities contributed 74.8% of total deliveries (34,087 orders).
* Motorcycles accounted for 58.0% of all deliveries (26,429 orders).
* Festival periods increased average delivery time by approximately 78%.
* Traffic jam conditions resulted in 46.6% higher delivery times compared to low-traffic conditions.
* Foggy weather conditions recorded higher delivery times compared to sunny conditions.
